# Read & Rhyme Buddy

A cute, gamified English phonics and sight-word app for a preschool learner.

The app helps a young reader practise:

- single alphabet sounds
- consonant blends
- digraphs and trigraphs
- short and long vowels
- vowel teams
- r-controlled vowels
- diphthongs
- common endings and letter patterns
- high-frequency sight words
- tiny decodable stories

## Features

- Sound cards with example words and picture cues
- Browser speech audio for sounds, words, sight words, and stories
- Word Quest quiz mode
- Sight Word shelf with 100 high-frequency words
- Story Maker with short read-aloud practice
- XP, stars, mastered sounds, and daily quest goals
- Static files that can be shipped on GitHub Pages, Vercel, Netlify, or Cloudflare Pages

## Run Locally

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Notes

Audio uses the browser Web Speech API, so available voices depend on the device and browser.
